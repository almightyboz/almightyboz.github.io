What is object_oriented programming?

A software paradigm that operates as a collection of discrete cooperating objects, instead of functions or commands. Each object is a "machine" in iteslf that can take, process, and send information to other objects. It's popularity for large-scale software engineering has roots in how it creates flexible and easy to maintain code.
